# Duet UX sprint — competitive conclusions (2026-07-16)

## Implemented this sprint
- Nav collapsed: **Today · Cal · Lists · More**
- Default app home = **Today** command center
- Natural language quick add (+ FAB)
- Event editor: location, notes, recurrence, reminders, comments
- Agenda + search
